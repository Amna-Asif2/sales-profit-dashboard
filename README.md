##  Sales & Profit Dashboard

This project is a **Sales & Profit Dashboard** that provides a clear view of business performance across products, categories, and months. It helps turn raw sales data into meaningful insights for better decision-making.

---

## 🚀 Project Overview

The dashboard analyzes sales data to identify:

* Top-performing product categories
* Monthly sales trends
* Profit-driving products
* Seasonal patterns in sales

This project demonstrates how data visualization can support business strategy and planning.

---

## 📈 Key Insights

* Electronics category shows strong performance
* Highest sales recorded in January and May 
* Cameras and Monitors generate the most profit


## 🛠️ Tools & Skills Used

* Data Cleaning
* Data Analysis
* Data Visualization
* Dashboard Design

*Excel*

---

## 📂 Dataset

The dataset includes sales records with:

* Order date
* Product name
* Category
* Region
* Quantity
* Sales 
* Profit

The goal of this project is to practice:
